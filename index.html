Service

ヒアリング

HEARING

お客様の理想にぴったりなWebサイトを制作するために、まずはWebサイトの目的や購入してもらいたい商品への導線、ビジネスのゴールについて詳しくお伺いします。丁寧なヒアリングを通じて、現状の課題を明確にし、効果的なプランをご提案します。

デザイン

DESIGN

ヒアリングでお伺いした内容をもとに、お客様の目的に合った効果的なデザインをご提案いたします。「見やすさ・使いやすさ」を意識し、何度でも訪れたくなるサイト作りを心がけています。
使用ソフト:Photoshop、Illustrator、Figma、Canva

ライティング

WRITING

ホームページやブログをはじめ、キャッチコピー、商品紹介、コラムなど、お客様のご要望に合わせた文章を丁寧に構成し、目的に応じた記事をお届けします。これまでに、SNS総フォロワー8.5万人を誇るインフルエンサーのSNS発信を2年以上サポートしてきた実績があります。この経験を活かし、読者の心を掴む記事をご提案します。

コーディング

CODING

HTML/CSSを使用し、魅力的で使いやすいWebサイトをお作りいたします。デザインや文章のご要望を丁寧にヒアリングし、お客様のアイデアを具体的な形にしてお届けします。PCサイトはもちろん、スマートフォンなど、あらゆるデバイスに最適化されたレスポンシブデザインにも対応いたします。
使用ソフト:Dreamweaver、Visual Studio Code
